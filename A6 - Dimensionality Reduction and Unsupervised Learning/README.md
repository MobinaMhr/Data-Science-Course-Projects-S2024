## How to Run

1. Create a virtual environment and activate it:

    Linux:

    ```bash
    python3 -m venv venv
    source venv/bin/activate
    ```

    Windows:

    ```bat
    python -m venv venv
    venv\Scripts\activate
    ```

2. Install the requirements:

    ```bash
    pip install -r requirements.txt
    ```
